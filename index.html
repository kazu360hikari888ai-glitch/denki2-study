📐 重要公式・数値まとめ

⚡ 電気回路の基本

オームの法則：V = I × R
電力：P = V×I = V²/R = I²R
熱量：Q = P×t [J]
1kWh = 3,600,000 J = 3.6MJ

🔌 インピーダンス・力率

直列Z：√(R²+(XL−XC)²)
リアクタンス：XL = 2πfL
容量性：XC = 1/(2πfC)
力率：cosφ = P / (V×I)

🏠 配電・電圧降下

単相2線：e = 2IR
単相3線：e = IR（片線分）
三相電力：P = √3×V×I×cosφ
同期速度：Ns = 120f/p

📏 施工の重要数値

ケーブル支持間隔：2m以下
合成樹脂管支持：1.5m以下
金属管曲げ半径：管内径×6倍以上
地中埋設（重量物）：1.2m以上

🛡 接地工事の種類

A種：10Ω以下（高圧機器）
B種：計算値以下（変圧器中性点）
C種：10Ω以下（300V超低圧）
D種：100Ω以下（300V以下）

📋 絶縁抵抗の最小値

対地電圧150V以下：0.1MΩ以上
150V超300V以下：0.2MΩ以上
300V超：0.4MΩ以上
測定器：500Vメガー（低圧）

🔑 分岐回路のコンセント

15A分岐：15A以下のコンセント
20A（CB）：20A以下
30A分岐：20A〜30A
40・50A：定格に合ったもの

🎨 電線の色・極性

接地側（N）：白色
非接地側（L）：黒・赤・青
接地線（E）：緑または緑/黄
コンセント長スロット：接地側（白）
